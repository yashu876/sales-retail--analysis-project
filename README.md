#  SQL Retail Sales Analysis

## Project Overview

This project analyzes retail sales data using SQL to understand business performance and generate useful insights. The analysis focuses on sales trends, customer behavior, and store performance.

## Dataset

The dataset contains retail transaction data including information about customers, stores, products, sales amount, quantity, discount, and transaction dates.

## Project Goal

The goal of this project is to use SQL to analyze retail data and extract insights that can help businesses understand their sales performance and make better decisions.

## Tools Used

* SQL
* GitHub


## Skills Demonstrated

* SQL Query Writing
* Data Analysis
* Data Cleaning
* Data Exploration

## Conclusion

This project shows how SQL can be used to analyze retail sales data and convert raw data into meaningful business insights.
